# Cómo instalar Meta Ads Diagnostic — 4 partes

Claude Code tiene un límite de ~50,000 caracteres por mensaje. Por eso la instalación se divide en **4 archivos**, que debes pegar en orden.

## Orden exacto

1. Abre `INSTALAR-parte-1-de-4.md` → Cmd+A → Cmd+C → pega en Claude Code → Enter. Espera a que Claude diga "Parte 1 completa".
2. Abre `INSTALAR-parte-2-de-4.md` → Cmd+A → Cmd+C → pega en **el mismo chat** → Enter. Espera "Parte 2 completa".
3. Abre `INSTALAR-parte-3-de-4.md` → Cmd+A → Cmd+C → pega → Enter. Espera "Parte 3 completa".
4. Abre `INSTALAR-parte-4-de-4.md` → Cmd+A → Cmd+C → pega → Enter. Claude muestra el mensaje de instalación completa.

## Qué queda instalado

- Proyecto en: `~/meta-ads-diagnostic/`
- Skill en: `~/.claude/commands/meta-ads-diagnostic.md`
- Logos de los 3 casinos principales (bmx, bet4-br, bet4-pe)

## Cómo usarlo después

Escribe `/meta-ads-diagnostic` en Claude Code y sigue las instrucciones en pantalla.
